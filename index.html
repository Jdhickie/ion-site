01

Meals on Wheels SaaS

A rural Irish Meals on Wheels branch was running client records, weekly meal plans, driver routes and billing across spreadsheets, paper sheets and WhatsApp — fragile, slow, and impossible to hand over.

Built a purpose-built operations platform covering client management, week planner, delivery sheets, driver routes, billing and cancellation tracking. In pilot with Glin Meals on Wheels — core features live, additional modules rolling out. Designed to scale to other Irish branches.

SaaS Operations Non-Profit React Supabase
02

AI Lead Response System

Freelance enquiries either get generic autoresponders (feels lazy) or wait days for a reply (feels unresponsive). Neither is good for the lead or the operator.

Built a system that reads each incoming enquiry, drafts a personalised reply with Claude based on what the person actually asked, and sends it automatically. Every lead is stored with its draft in Supabase. A kill-switch toggle lets the operator flip back to a manual approval flow at any time.

Live on this website — the contact form below runs on it.

AI Automation Claude API Vercel Supabase
03

Financial Dashboard

Tracking real income and expenditure across multiple payment sources as a solo operator is either done in spreadsheets (tedious, out-of-date) or outsourced (expensive, slow feedback loop).

Built a dashboard that pulls transaction data from Stripe, combines it with manually logged entries in Supabase, and surfaces income, expenditure and cashflow trends in one view. Query-based so new cuts can be added as questions come up.

Running as an internal tool for Íon. Adaptable for other small operators who want financial visibility without accounting-software overhead.

Fintech Stripe Dashboards Supabase
04

Diesel Price Tracker

Irish fleet operators and long-distance drivers have no easy way to see where fuel is cheaper along a route — prices vary meaningfully between stations and regions.

Built a community-powered fuel price map for Ireland with a route planner layer, so operators can compare prices across stations on their actual journey rather than just nearest-location.

Prototype built, not yet deployed. Intended as a free consumer tool with a premium layer for logistics operators.
